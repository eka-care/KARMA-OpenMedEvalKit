## Why KARMA?

KARMA is a comprehensive, high-performance evaluation framework for building medical AI benchmarks with Python 3.12+ based on standard PyTorch models. Designed for researchers, developers, and healthcare organizations who need reliable evaluation of medical AI systems.

## Quick Start

Get started with KARMA in minutes:

```bash
# Install KARMA
git clone https://github.com/eka-care/KARMA-OpenMedEvalKit.git
cd KARMA-OpenMedEvalKit && uv sync

# Run your first evaluation
karma eval --model "Qwen/Qwen3-0.6B" --datasets pubmedqa --max-samples 3
```

## Key Features

- **Registry-Based Architecture**: Auto-discovery of models, datasets, and metrics
- **Smart Caching**: DuckDB and DynamoDB backends for faster re-evaluations  
- **Extensible Design**: Easy integration of custom models, datasets, and metrics
- **Rich CLI**: Beautiful progress bars, formatted outputs, and comprehensive help
- **Standards-Based**: Built on PyTorch and HuggingFace Transformers
